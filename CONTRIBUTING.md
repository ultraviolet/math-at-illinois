# How to Contribute

Thank you for your interest to contributing! 

All you need is a [GitHub account](https://github.com). Contributions are managed through our [GitHub repository](https://github.com/3nya/math-at-illinois), where you can directly submit changes or propose improvements.

All the content of the website is located in the `/content` folder. Within `/content` there are subfolders referring to specific topics such as `/classes`, `/research`, etc, then the md files corresponding to pages on the website. 

General steps:
1. **Fork the repository** if prompted. This creates a copy of the repository in your GitHub account.
2. **Make your changes** directly in the GitHub editor or clone the repository to work locally. Steps to build the project is in the [README](https://github.com/3nya/math-at-illinois/blob/main/README.md)
3. **Commit your changes**
4. **Create a Pull Request (PR)** on the GitHub website. 
5. **Review process**: A maintainer will review your changes. Please feel free to ping on the Discord server :D

# Specific Contributions

## Fixing an Existing File
Just edit the file directly and follow the steps above.

## Adding non course content
For example a page on research/extracurricular/etc

Currently there's no specific template for adding non-course content. However, please keep the pages informative and descriptive. If you're unsure about something, please ask within the #wiki channel on [Discord](https://discord.gg/Fds22YywnA).
